# FridgeChef 프로젝트 설정 가이드

FridgeChef(내 냉장고 속 AI 셰프)는 Google Cloud, Google Sheets, Gemini API를 사용하는 프로젝트입니다.  
아래 문서를 순서대로 참고해 주세요.

1. Cloud/API 설정: `CLOUD_CONFIG.md`
2. 스프레드시트 설정: `SHEETS_CONFIG.md`
3. 환경 변수(.env) 형식: `ENV_EXAMPLE.md`
4. 로컬 실행 및 터미널 명령어: `TERMINAL_GUIDE.md`

이 네 파일만 따라 하면, 로컬에서 백엔드를 실행하고 앱을 테스트할 수 있습니다.
