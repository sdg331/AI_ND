# FridgeChef (내 냉장고 속 AI 셰프)


## 터미널 명령어
```text
[venv활성화] .\venv\Scripts\Activate.ps1
[백엔드진입] cd backend
[실행] python app.py
```

## 개요

FridgeChef는 인공지능(AI)과 멀티모달(이미지+텍스트) 분석 기술을 활용해,  
냉장고 속 재료와 나의 요리 프로필을 기반으로 맞춤 레시피를 생성해 주는 모바일 쿠킹 솔루션입니다.

단순 레시피 검색이 아니라,
- 냉장고 사진/영수증/텍스트로 재료를 쉽게 등록하고,
- 선호/비선호 재료, 알레르기, 보유 조리도구, 식단 목표를 반영한 레시피를 생성하며,
- (확장) 요리 중 발생하는 돌발 상황을 AI 챗봇이 실시간으로 도와주는
“내 손안의 AI 셰프 + 요리 코치”를 지향합니다.

## 주요 기능 및 페이지 (MVP 기준)

### 1. 나의 냉장고 (Fridge)

- 냉장고 내부 사진 촬영 → Vision AI로 재료 인식
- 마트 영수증 촬영 → OCR로 품목 추출
- 직접 텍스트 입력으로 재료 추가
- 등록된 재료 목록 조회/수정/삭제
- (확장) 유통기한 기준 정렬 및 임박 재료 강조

### 2. 프로필 (Profile)

- 선호/비선호 재료 설정
- 알레르기 정보 등록 (절대 사용 금지 재료)
- 보유 조리도구 설정 (에어프라이어, 전자레인지, 오븐 등)
- 식단/건강 목표 설정 (다이어트, 비건, 저염식 등)
- 선호 요리 난이도/스타일 설정 (초간단, 한식 위주 등)
- 이 정보들이 AI 레시피 생성 시 필터/가중치/스타일 가이드로 반영

### 3. AI 레시피 생성 (Recipes)

- “내 냉장고 + 내 프로필” 정보를 바탕으로 레시피 생성
- 레시피 카드 리스트:
  - 제목, 예상 조리 시간, 난이도, 사용 재료 요약
- 레시피 상세:
  - 냉장고 재료 vs 추가로 필요한 재료 구분
  - 단계별 조리법
  - (확장) 간단 영양 정보, 팁

### 4. 쿠킹 모드 & (P1) AI 쿠킹 어시스턴트

- 단계별 조리 가이드를 한 화면씩 보여주는 “쿠킹 모드”
- (P1) 요리 중:
  - “간이 짜요”, “이 재료 대신 뭐 써요?” 같은 질문을 하면
  - AI 쿠킹 어시스턴트가 실시간으로 해결책/대체 재료를 제안

## 페이지 상세 (모바일 앱 기준)

### 홈(Home) 화면

- 오늘의 냉장고 요약(재료 수, 임박 재료 하이라이트)
- “내 재료로 레시피 만들기” 메인 버튼
- 최근 생성한 레시피 카드 목록

### 나의 냉장고(Fridge) 화면

- 재료 등록 방식 3가지 카드:
  - 냉장고 사진 촬영
  - 영수증 스캔
  - 직접 입력
- 등록된 재료 리스트(이름, 수량, 유통기한)
- 정렬/필터(유통기한 순, 카테고리별 등)

### 레시피(Recipes) 화면

- “레시피 생성하기” 버튼
- AI가 생성한 레시피 카드 리스트 (제목, 시간, 난이도)
- 레시피 상세 → “요리 시작” → 쿠킹 모드 진입

### 프로필(Profile) 화면

- 설정 중심 프로필:
  - 선호/비선호 재료
  - 알레르기
  - 보유 조리도구
  - 식단/건강 목표
  - 선호 난이도/스타일
- AI 레시피 생성 시 이 설정 정보가 반영됨
